Affirmation App
-------------
This App show image in a scrollable list.

Concepts Covered
-----------------
- Create model class Affirmation for image and string.
- Create data class DataSource for load all list of affirmation.
- Use of Column, Card.
- Using LazyColumn for showing scrollable list.
